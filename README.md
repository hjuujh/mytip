## 프로젝트_설명
 *MY TIP은 티켓 사진을 이용하여 구글 클라우드 비전 API으로 티켓의 문자를 인식하고 그 인식 결과를 가지고 편하게 후기를 쓸 수 있는 앱 입니다.
 *내 손 안에서 편하게 티켓과 후기를 관리하자는 생각에서 만들었습니다.
